# Backend (Django)

This folder is reserved for the Django backend.

## Suggested setup
```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# create project
# django-admin startproject core .
# python manage.py migrate
# python manage.py runserver
```

## Structure (planned)
- `core/` — Django project
- `apps/` — Django apps
- `manage.py`
